=== 更新链接创建

* Codebox 查看器：Codebox 现在集成在链接创建过程中。您可以在 Workload Factory 中的 Codebox 查看和复制 CloudFormation 模板，然后重定向到 AWS 执行操作。
* 所需权限：现在可以从 Workload Factory 中的创建链接向导查看和复制在 AWS CloudFormation 中执行链接创建所需的权限。
* 支持手动创建链接：此功能允许通过手动注册链接ARN在AWS CloudFormation中独立创建。当安全或开发运营团队协助创建链接时、这将非常有用。
