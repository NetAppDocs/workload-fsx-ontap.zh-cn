= 在NetApp Workload Factory中管理ONTAP文件系统标签的 FSx
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
标签可以帮助你对资源进行分类。在NetApp Workload Factory中，您可以随时添加、编辑和删除文件系统的标签。

.步骤
. 使用其中一个登录link:https://docs.netapp.com/us-en/workload-setup-admin/console-experiences.html["控制台体验"^]。
. 选择菜单 image:workloads-menu-icon.png["汉堡菜单图标用于导航至存储、EDA、AI、数据库、VMware 和管理等工作负载。"] 然后选择“存储”。
. 从存储菜单中，选择 *FSx for ONTAP*。
. 从 *FSx for ONTAP* 中选择要更新的文件系统的操作菜单，然后选择 *管理文件系统标签*。
. 在“管理文件系统标签”对话框中，根据需要添加、编辑或删除标签。
+
文件系统最多可以应用 50 个标签。

. 选择 * 应用 * 。
